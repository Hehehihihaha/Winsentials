Using Winsentials is simple. The main screen organizes tweaks into sections:

- Privacy  
- Optimization

1. Select the tweak you want.  
2. Read the explanation.  
3. Click `Apply`.

The app automatically backs up your current settings before applying any tweak. This backup lets you restore your PC if you do not like the results.

## 🔧 Troubleshooting Common Issues

If you face problems, try these steps:

- **Installer won’t start:** Right-click the installer and choose `Run as administrator`.  
- **App crashes or freezes:** Restart your computer and open Winsentials again.  
- **Tweak has no effect:** Make sure you restarted Windows if the tweak asks for it.  
- **Windows reports an error on start:** Use the backup feature in Winsentials to undo recent changes.  

If problems continue, visit the issues section on the GitHub page.
